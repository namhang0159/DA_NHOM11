##  Giới thiệu
Website quản lý shop quần áo trực tuyến gồm giao diện trang quản trị.  
Hệ thống được xây dựng theo mô hình **MVC**, sử dụng **React + Node.js + MySQL**.

---

##  Công nghệ
- Frontend: ReactJS, Next.js, Ant Design
- Backend: Node.js, Express, Sequelize
- Database: MySQL
- Khác: JWT, Multer, RESTful API

---


---

### Backend

### FrontEnd
